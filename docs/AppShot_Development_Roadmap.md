# AppShot 開發進度待辦清單

_依據 AppShot MVP 規劃、技術實現指南及 UI/UX 設計規範制定_

## 專案概述

AppShot 是一款極簡的 App Store 截圖生成工具，無需註冊即可使用，幫助 iOS 應用開發者快速生成符合上架規範的應用截圖。本專案採用純前端架構，所有圖像處理都在使用者瀏覽器中完成，不需要後端服務。

## 開發週期

本開發計劃依循 4 週開發週期，從環境設置到最終部署，確保高效穩定地交付 MVP 版本。
